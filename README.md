# APEX Academy Website

Sitio estático inicial para APEX Academy.

## Contenido

- `index.html`: landing page principal.
- `assets/styles.css`: estilos visuales responsive.
- `assets/script.js`: menú móvil, animaciones y año automático.
- `assets/logo-apex.svg`: logo vectorial editable.
- `assets/favicon.svg`: favicon.
- `assets/apex-og.svg`: imagen Open Graph.
- `netlify.toml`: configuración para Netlify.
- `vercel.json`: configuración para Vercel.
- `404.html`: página de error.
- `robots.txt`: indexación básica.

## Publicación rápida

### Opción 1: Netlify

1. Entrar a Netlify.
2. Crear un nuevo sitio.
3. Subir esta carpeta completa comprimida o conectarla a GitHub.
4. Publicar. No requiere build command. Publish directory: `.`

### Opción 2: Vercel

1. Crear un proyecto en Vercel.
2. Importar el repositorio desde GitHub o subir los archivos.
3. Framework: Other.
4. Output directory: `.`
5. Deploy.

### Opción 3: GitHub Pages

1. Crear un repositorio, por ejemplo `apex-academy-site`.
2. Subir todos los archivos.
3. Ir a Settings > Pages.
4. Source: Deploy from branch.
5. Branch: `main` / root.
6. Guardar y esperar la URL publicada.

## Antes de publicar oficialmente

- Reemplazar `mailto:contacto@apexacademy.local` por el correo real.
- Actualizar `og:url` cuando exista dominio oficial.
- Crear `sitemap.xml` con el dominio final.
- Agregar documento descargable APEX-000 cuando esté validado como público.
